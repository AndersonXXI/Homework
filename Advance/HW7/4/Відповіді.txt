Серіалізація/десеріалізація:
Серіалізація - це процес перетворення об'єктів чи даних у структуру, що може бути збережена або передана через мережу у вигляді послідовності байтів або тексту. Десеріалізація - це процес відновлення серіалізованих даних з їхнього байтового або текстового представлення в об'єкти чи даних.

Маршаллінг/демаршаллінг:
Маршаллінг - це процес перетворення об'єктів в XML-подібне представлення. Демаршаллінг - це процес відновлення об'єктів з XML-подібного представлення.

XML (Extensible Markup Language):
XML - це розширюваний мовний формат для представлення даних у структурованому вигляді. Використовується для обміну даними між різними системами та мовами програмування. Різниця між XML та HTML полягає у їхньому призначенні: XML призначений для структурованого представлення даних, тоді як HTML використовується для створення сторінок веб-сайтів.

Парсери в XML:
Існує кілька типів парсерів XML:
a. DOM-парсер: Завантажує весь XML-документ у пам'ять та створює дерево DOM (Document Object Model). Дозволяє виконувати редагування та навігацію по документу.
b. SAX-парсер: Парсер XML-документу по одному елементу за раз. Дозволяє обробляти великі XML-файли, оскільки не потребує зберігання всього документу в пам'яті.
c. StAX-парсер: Забезпечує покрокове читання та запис XML-документів. Поєднує переваги DOM та SAX парсерів.

JSON (JavaScript Object Notation):
JSON - це легкий формат обміну даними, який використовується для передачі структурованих даних між різними системами. Використовується для об'єктів, масивів, чисел, рядків, булевих значень та значення null. JSON є популярним у веб-розробці, оскільки легко зрозуміти і робити парсинг в браузері.

JAXB (Java Architecture for XML Binding):
JAXB - це технологія Java, яка надає спосіб зв'язування даних між XML-документами та об'єктами Java. Дозволяє перетворювати XML в об'єкти Java (маршалінг) та об'єкти Java в XML (демаршаллінг). JAXB є частиною стандартного пакету Java SE та допомагає спростити роботу з XML в Java.
